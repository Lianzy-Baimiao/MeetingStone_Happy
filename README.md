#### 介绍
##### 魔兽世界集合石插件魔改功能：
 1. 1~2倍UI缩放
 2. 设置/其他UI优化 
 3. 屏蔽队长、举报队伍 
 4. 双击进组/自动进组/自动邀请 
 5. 申请列表、队伍列表显示优化
 6. 创建队伍最低申请分数设置
 7. 跨阵营、显示职业图标、查看队伍专精、申请人显示职业图标
 8. 可供快速筛选及创建的当前版本大秘境信息

> **职业图标显示（代码段来源于NDui，与Wind工具箱预组建队伍美化冲突，需二选一，可在设置里关闭！！！）**
